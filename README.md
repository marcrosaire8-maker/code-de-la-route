# Code de la Route Bénin — Jeu interactif ANaTT (Permis B)

Application web autonome pour s'entraîner au code de la route béninois (progression en 12 000 étapes, secteurs thématiques, timer, vies, score par séries de 20).

## Lancer en local

**Prérequis :** Node.js (npm)

1. Installer les dépendances :
   `npm install`
2. Démarrer le serveur de dev :
   `npm run dev`

## Build (production)

`npm run build`
